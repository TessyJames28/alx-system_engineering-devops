Starting ALX system engineering devops with basic shell scripting
Different script perform the following tasks
pwd - print current working directory
ls - list content of current working directory
cd ~ - change to home directory
ls -l - prints content of directory in a long list format
ls -la - print content of directory in a long list format with hidden files
ls -nla - prints directory content with UID and GID displayed numerically with hidden files
mkdir - creates a new directory
mv - moves file to a new directory
rm - delete file
rmdir - delete directory
